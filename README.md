How to use game
1. run follow command in cmd "php artisan StartGame"
2. input TeamA values (Example: 30, 100, 20, 50, 40)
3. input TeamB values (Example: 30, 100, 20, 50, 40)

TeamA will win if all members level more than TeamB or equal.
